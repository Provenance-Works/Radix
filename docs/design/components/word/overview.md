# Word Module: Overview

Version: 0.1.0-draft
Last updated: 2026-03-07

## 1. 概要
`Word` モジュールは、Radixシステムプログラミングライブラリの最下層に位置する基礎コンポーネントです。固定長整数（符号なし・符号あり）およびプラットフォーム依存サイズ（ポインタ幅）の整数型を提供し、それぞれの型に対する算術演算と基本ビット演算（Bitモジュールとの連携）を定義します。

## 2. 責務
- C言語の基本型（`uint8_t`, `int32_t`, `size_t` 等）と同一のメモリ表現・計算セマンティクスを持つデータ型の提供。
- Lean 4の組み込み `UInt` 型をラップすることによる、**ゼロコスト抽象化（NFR-002）**の達成。
- Mathlibの `BitVec` との同値性証明による、厳密な形式検証モデル（Layer 3）の提供。
- ユーザーが用途に合わせて選択できる、証明付き（Tier 1）およびモード付き（Tier 2）算術APIの提供（FR-001.2）。

## 3. 依存関係
- **Mathlib (`Data.BitVec`)**: 仕様定義（Layer 3）および証明にのみ使用。実行時の計算（Layer 2）には依存しません。
- 他のRadix内モジュールへの依存はありません（依存グラフの最根幹に位置します）。

## 4. 内部構造
`Word` モジュールは以下のサブコンポーネントに分割されます。
- `Word.Spec`: 算術・ビット演算の純粋な数学的仕様（Layer 3）。
- `Word.UInt`: 符号なし固定長整数の定義。
- `Word.Int`: 符号あり固定長整数の定義（内部的にUIntNを2の補数として解釈）。
- `Word.Arith`: Wrapping, Saturating, Checked, Overflowingの各算術オペレーションの実装。
- `Word.Conv`: 異なるビット幅間、符号間のキャスト・拡張・切り捨て処理。
- `Word.Lemmas`: 各種操作の代数的性質や、`BitVec` との完全な同値性の証明群。
